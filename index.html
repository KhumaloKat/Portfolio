Drone Hand Gesture Control

  • This project enables control of a Tello drone using hand gestures, processed by computer vision algorithms. The system utilizes Python, cvzone, and MediaPipe to detect and interpret hand movements, translating them into drone commands. The user can operate the drone seamlessly without a physical controller, enhancing interaction and ease of control.

Drone Face Following

  • This project equips a Tello drone with face-tracking capabilities. By utilizing facial recognition algorithms through Python and cvzone, the drone autonomously follows a person based on their facial features. This makes it useful for monitoring, as the drone can stay focused on a subject’s face, maintaining a smooth following experience.

Drone Body Following

  • The Drone Body Following project allows a Tello drone to track and follow a person by detecting and locking onto their body. Face door lock

  • The Face Door Lock project integrates facial recognition technology to secure access to doors. Using Python with cvzone and MediaPipe libraries for face detection and recognition, combined with Arduino for the locking mechanism, this system authenticates users based on facial features. The solution enhances security by ensuring only authorized individuals can unlock the door, providing a modern alternative to traditional key-based locks.
